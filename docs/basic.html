实验研究的设计及分析

第 2 章 几种基本的实验设计

有四种基本的实验设计：

  1. 单因素完全随机实验设计
  2. 单因素随机区组实验设计
  3. 单因素拉丁方实验设计
  4. 单因素重复测量实验设计

2.1 单因素完全随机实验设计

单因素完全随机实验，适用于研究中只有一个自变量，自变量有多个水平。基本方法是把被试随机分配给处理的各个水平，每个被试只接受一个水平的处理。

完全随机实验设计使用随机化控制误差变异。

单因素完全随机实验设计模型：

\[Y_{ij} = \mu + \alpha_j + \epsilon_{i(j)} \]

其中：

  1. \(Y_{ij}\) 表示被试i在处理水平j上的分数；
  2. \(\mu\) 表示总体平均值，或真值；
  3. \(\alpha_j\) 表示水平j的处理效应；
  4. \(\epsilon_{i(j)}\) 表示误差效应，是一个符合正态分布的随机变量。

从这个模型可以看出，因变量的总变异是由两部分组成，即实验处理引起的变异和误差引起的变异。

阅读理解随着生字密度的增加而下降

数据处理，通过SPSS软件中的单因素方差分析实现。通过F检验，检验实验处理带来的效应是否不同于实验误差，如果差异达到一定的统计显著水平，表明处理的效应是存在的。

在这种实验实际中，组内方差，指所有不能用实验处理解释的变异，它可能包括被试个体差异、无关变异和实验误差。

这种设计的缺点是，组内变异并非全部由随机误差组成，实际上被试个体差异带来的无关变异时存在的，并且混杂在组内变异中，导致F比率的分母项变大，从而使得实验较为不敏感。

2.2 单因素随机区组实验设计

随机区组设计使用区组方法分离出由无关变量引起的变异，从而减少误差变异。单因素随机区组实验设计模型如下：

\[Y_{ij} = \mu + \alpha_j + \pi_i + \epsilon_{i(j)} \]

其中：

  1. \(Y_{ij}\) 表示被试i在处理水平j上的分数；
  2. \(\mu\) 表示总体平均值，或真值；
  3. \(\alpha_j\) 表示水平j的处理效应；
  4. \(\pi_i\) 表示区组效应；
  5. \(\epsilon_{i(j)}\) 表示误差效应。

单因素随机区组设计适用于：

  1. 研究中有一个自变量（有两个以上水平）；
  2. 研究中还有一个无关变量（有两个以上水平）；
  3. 自变量的水平与无关变量的水平之间没有交互作用。

阅读理解随着生字密度的增加而下降，智力作为无关变量，通过实验设计将它的效应分离出去。

在SPSS软件中的统计分析：选择分析->一般线性模型->单变量菜单，将无关变量和自变量放在固定因子中，打开模型对话框，选择构建部，将构建项设置为主效应，然后将无关变量和自变量加入到模型中，点击继续和确定。

2.3 单因素拉丁方实验设计

拉丁方实验扩展了随机区组实验设计的原则，可以分离出两个无关变量的效应。一个无关变量的水平在横行分配，另一个无关变量的水平在纵列分配，自变量的水平则分配给方格的每个单元。

需要满足下列条件：

  1. 研究中有一个两个以上水平的自变量。
  2. 有两个带有两个以上水平的无关变量。
  3. 自变量的水平数与无关变量的水平数相等。
  4. 处理水平与无关变量水平之间没有交互作用。
  5. 随机分配处理水平给拉丁方格，每个处理水平仅在每行、每列出现一次。
  6. 每个方格单元中分配1个或多个被试接受处理。

单因素随机拉丁方实验设计模型如下：

\[Y_{ijkl} = \mu + \alpha_j + \beta_k + \gamma_l + \epsilon_{pooled} \]

其中：

  1. \(Y_{ij}\) 表示被试i在处理水平j上的分数；
  2. \(\mu\) 表示总体平均值，或真值；
  3. \(\alpha_j\) 表示水平j的处理效应；
  4. \(\beta_k\) 表示水平k的无关变量B的变异；
  5. \(\gamma_l\) 表示水平l的无关变量C的变异；
  6. \(\epsilon_{pooled}\) 表示误差变异。

2.4 单因素重复测量实验设计

个体差异会引起无关变异，而控制个体差异的一个有效方法就是重复测量实验设计，也叫被试内设计。

重复测量实验设计的基本方法是：实验中每个被试接受所有的处理水平。这种实验设计的目的是利用被试自己做控制，使被试的各方面特点在所有的处理中保持恒定，以最大限度地控制由被试的个体差异带来的变异。

使用条件：

  1. 当若干处理水平连续实施给统一被试时，被试接受前面的处理对接受后面的处理没有长期影响。
  2. 需要对顺序效应、平衡效应进行平衡。

数据分析：数据在录入时，一个水平一个变量。而且在SPSS中，单因素重复测量实验没有对应的菜单命令。例如下方的数据文件，就需要如下代码(舒华 2015)：

MANOVA a1 a2 a3 a4
    /WSFACTORS=a(4)
    /PRINT = CELLINFO(MEANS)
    /DESIGN.

其中：

  1. MANOVA a1 a2 a3 a4表示重复测量的因素有a1 a2 a3 a4四个水平。
  2. /WSFACTORS=a(4)表示被试内因素的名称及水平。
  3. /PRINT = CELLINFO(MEANS) 表示计算每个实验单元的平均数和标准差。
  4. /DESIGN. 是所有方差分析程序中的最后一个分命令，将保证程序完成一个完整的方差分析。
